# ✍️ Contributing to Seurausten talo

Seurausten talo ei ole mielipideprojekti eikä avoin ideaseinä.
Se on institutionaalinen muistirakenne, jonka tarkoitus on tehdä näkyväksi
päätösten pitkät ja kertautuvat seuraukset.

Kontribuutiot ovat tervetulleita,
mutta vain, jos ne noudattavat tätä peruslähtökohtaa.


## Ennen kuin kontribuoit

Kysy itseltäsi:

- Kuvaanko *päätöstä*, en pelkkää tapahtumaa?
- Sisältääkö tämä *ajallisen viiveen*?
- Näyttääkö tämä *seurauksia*, ei vain vaikutuksia?
- Vältänkö moraalista tuomiota ja jälkikäteistä viisautta?

Jos vastaus on useimmiten “kyllä”, jatka.


## Mitä voi kontribuoida

### 1. Case-tapauksia (ensisijainen kontribuutio)

Uudet tapaukset ovat projektin ydin.

Hyvä case:
- kuvaa institutionaalista, teknologista, hallinnollista tai kulttuurista päätöstä
- on toistettavissa (ei täysin ainutkertainen poikkeus)
- voidaan kuvata päätöksen elinkaaren (1–6) kautta
- tunnistaa vaikutuskerrokset (esim. terveys, informaatio, luottamus, hallinto)

Huono case:
- etsii syyllisiä
- keskittyy yksittäisiin henkilöihin
- ohittaa aikajänteen
- pyrkii todistamaan ennalta valittua johtopäätöstä


### 2. Rakenteelliset parannukset

Voit kontribuoida myös:
- datamallin selkeyttämiseen
- visualisointien luettavuuteen
- dokumentaation täsmennykseen

Tekninen eleganssi on toissijaista
rakenteellisen ymmärrettävyyden jälkeen.


## Case-rakenne (lyhyesti)

Kaikkien tapausten tulee noudattaa yhteistä rakennetta
(data-cases.js):

- id
- title
- domain
- impactLayers
- populationExposure
- conflictMode
- lifecycle (1–6)

Tämä yhteismitallisuus on olennainen.
Ilman sitä vertailevuus katoaa.


## Sävy ja kieli

Seurausten talon kieli on:

- kuvailevaa, ei normatiivista
- täsmällistä, ei iskulausemaista
- hidasta, ei reaktiivista

Vältä:
- moraalista paatosta
- poliittista kampanjakieltä
- ironiaa tai sarkasmia
- yksinkertaistavia vastakkainasetteluja


## Pull request -käytännöt

- Yksi pull request = yksi selkeä muutos
- Kuvaa *mikä rakenne paranee*, ei miksi olet oikeassa
- Varaudu siihen, että kontribuutio voidaan hylätä
  ilman että se on henkilökohtainen arvio

Seurausten talo ei kasva nopeasti.
Se kasvaa harkiten.


## Keskustelu ja erimielisyys

Erimielisyydet ovat odotettuja.
Niitä käsitellään:

- rakenteen tasolla
- aikajänteen tasolla
- vaikutusketjujen tasolla

Ei:
- moraalisena kamppailuna
- poliittisena väittelynä
- identiteettien kautta


## Lopuksi

Kontribuoiminen Seurausten taloon
tarkoittaa osallistumista institutionaaliseen muistityöhön.

Se ei lupaa näkyvyyttä,
ei nopeaa vaikutusta,
eikä varmuutta siitä, että olet oikeassa.

Se lupaa vain,
että päätösten seuraukset pysyvät näkyvissä hieman pidempään.

Kiitos harkinnasta.
